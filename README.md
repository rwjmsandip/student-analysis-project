# Student Analysis System #

## Project Description
This project is a Python-based Student Data Analysis System
-Student management (Add, Update, Delete)
-Statistical analysis
-Probability calculations
-Data visualization using Matplotlib

## Features
-Add / Update / Delete students
-Mean, Median, Mode, Variance, Standard Deviation
-Probability analysis
-Histogram, Bar chart, Box plot, Scatter plot, Line chart

## Technologies Used
-Python
-Matplotlib
-Statistics module

## How to Run
python main.py
